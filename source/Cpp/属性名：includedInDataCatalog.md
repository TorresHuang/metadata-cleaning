# 属性名： schema:includedInDataCatalog

## 1. 描述
包含的数据分类。

## 2. 数据处理流程
### 2.1 新收割来的数据-数据处理流程：
  * 直接将数据存储进对应的 ___schema:includedInDataCatalog___ 字段中。

## 3. 标准规范
```json
"schema:includedInDataCatalog" : 
[
    "极地",
    {
        "@value" : "Polar region",
        "@language" : "en"
    }
],
```
